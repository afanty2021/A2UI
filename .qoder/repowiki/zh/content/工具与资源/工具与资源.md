## A2UI编辑器

A2UI编辑器是一个可视化工具，允许开发者通过拖拽组件的方式实时生成A2UI JSON格式的UI描述，用于调试和原型设计。该工具提供了一个直观的界面，用户可以通过上传图片或手绘草图来生成相应的UI结构。

要运行A2UI编辑器，需要满足以下前提条件：
- 拥有Gemini API密钥
- 安装Node.js

运行步骤如下：
1. 在编辑器目录下创建一个`.env`文件，并在其中设置`GEMINI_API_KEY`环境变量
2. 安装依赖：`npm i`
3. 启动开发服务器：`npm run dev`
4. 在浏览器中访问 http://localhost:5173/

编辑器支持两种输入模式：上传图片和手绘草图。用户可以在左侧面板中输入对期望UI的文本描述，然后点击"生成UI"按钮，右侧将实时显示生成的UI界面。此外，用户还可以切换到消息模式，查看生成的A2UI JSON消息，并可一键复制到剪贴板。

## A2UI检查器

A2UI检查器是一个用于捕获和解析A2UI消息流的工具，帮助开发者诊断和调试A2UI相关的问题。该工具允许用户直接输入A2UI JSON消息，并可视化地渲染出对应的UI界面，便于分析消息结构和验证渲染效果。

与编辑器不同，检查器不需要Gemini API密钥，仅需Node.js环境即可运行。使用方法如下：
1. 安装依赖：`npm i`
2. 启动开发服务器：`npm run dev`
3. 在浏览器中访问 http://localhost:5173/

用户可以在左侧的文本区域直接粘贴A2UI JSON消息，然后点击"生成UI"按钮，右侧将立即渲染出对应的UI界面。如果JSON格式有误，检查器会显示相应的错误信息。该工具特别适用于分析从生产环境捕获的A2UI消息流，或验证自定义组件的渲染效果。

## 示例项目

项目提供了`samples/`目录下的多个完整示例，从简单的组件展示到复杂的多代理协作系统，帮助开发者快速理解和使用A2UI。

### Gallery示例

位于`samples/client/angular/projects/gallery/`的Gallery示例是一个组件展示库，展示了A2UI支持的各种UI组件，包括列表、欢迎卡片和联系表单等。该示例通过代码直接创建A2UI表面（Surface），演示了如何使用A2UI构建常见的UI模式。

**Section sources**
- [samples/client/angular/projects/gallery/src/app/features/gallery/gallery.component.ts](file://samples/client/angular/projects/gallery/src/app/features/gallery/gallery.component.ts#L1-L247)

### Restaurant Finder示例

Restaurant Finder是一个餐厅查找和预订代理的示例，展示了如何使用Agent Development Kit (ADK)和A2A协议创建一个能够生成动态UI的代理。该代理可以根据用户查询生成包含餐厅列表和预订表单的交互式界面。

**Section sources**
- [samples/agent/adk/restaurant_finder/README.md](file://samples/agent/adk/restaurant_finder/README.md#L1-L40)

### Orchestrator示例

Orchestrator示例展示了一个更复杂的多代理协作系统，其中主代理（orchestrator）负责将请求路由到不同的专业子代理。系统包含三个子代理：联系人查找、餐厅查找和Rizzcharts数据分析。主代理根据用户请求的内容决定将任务分配给哪个子代理，并将生成的UI整合到统一的界面中。

## 社区支持

A2UI拥有活跃的社区支持，开发者可以通过多种渠道获取帮助、分享经验和贡献代码。

### GitHub Discussions

项目在GitHub上设有Discussions区域，是提问、分享想法和讨论A2UI相关话题的主要平台。开发者可以在这里寻求技术帮助、分享自己的项目或参与功能讨论。

### Discord频道

除了GitHub Discussions，项目还维护着一个Discord频道，提供实时交流的机会。开发者可以加入Discord与核心团队和其他社区成员直接互动。

### 贡献指南

A2UI是一个开源项目，欢迎各种形式的贡献，包括但不限于：
- 报告问题和bug
- 开发新的渲染器（如React、Vue等）
- 分享示例和教程
- 改进文档

更多关于如何贡献的信息，请参阅项目的CONTRIBUTING.md文件。
